<details>
  <summary>🎓 <b>College Placement Analytics & Management System</b></summary>
  <br />

  An enterprise-grade, 3NF normalized database system integrated with an interactive python dashboard for managing and analyzing college recruitment lifecycles.

  | Attribute | Specification |
  | :--- | :--- |
  | **Stack** | MySQL, Python, Streamlit, Pandas, Matplotlib, SQL Connector |
  | **Scale** | Seeded with 350+ Students, 50 Companies, 150 Jobs, 950+ Applications, 600+ Interviews |
  | **Performance** | Configured B-Tree indexes on query filtering paths (CGPA, application statuses) |
  | **Security** | Implemented parameterization to prevent SQL injections; isolated config using dotenv |
  | **Impact** | Automates funnel updates and screens eligible candidates in real-time |
  | **Repository** | [GitHub Link](https://github.com/madeehasheikh/college-placement-analytics-system) |

  #### Professional Project Summary
  CPAMS resolves data isolation and coordination issues in traditional college placement cells. It implements a 10-table relational schema designed up to 3NF. Features include database triggers (`trg_after_offer_insert`) to handle automatic application state transitions, views (`v_company_hiring_stats`) for real-time aggregation, and transaction-wrapped stored procedures (`sp_record_placement_acceptance`) to handle offer acceptances with rollback safety. The UI provides administrative command modules, student profile dossiers, and department analytics.
</details>
